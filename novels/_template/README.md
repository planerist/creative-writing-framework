# `novels/_template/` — проработанное демо-произведение

Это **жанрово-нейтральное произведение-пример** (*«Последний паром в Олдермонт»*, тихая
литературная драма). Оно существует, чтобы показать форму проекта, так чтобы вы увидели всю
машину в работе прежде, чем писать собственную книгу. **Чтобы начать настоящую книгу:**

```
cp -R novels/_template novels/<ваш-роман>
```

Затем сохраните структуру и замените содержимое.

## Что есть в этой папке

| Путь         | Что это                                                                                                   | Кто это читает/пишет |
| ------------ | --------------------------------------------------------------------------------------------------------- | --- |
| `premise.md` | Одностраничный бриф истории: логлайн, сама история, эмоциональный/телесный стержень, чем она НЕ является. | Вы это пишете; workflow читают это как ориентир. Не строгий канон. |
| `knowledge/` | **База знаний (KB)** — канон. См. `knowledge/README.md`.                                                  | Читается каждым workflow отрисовки/анализа; обновляется через `/sync-kb` (предлагает изменения для разбора и применения). |
| `chapters/`  | Рукопись и её артефакты по главам. См. `chapters/README.md`.                                              | Пишется через `/draft`, `/snapshot`, `/meta`; читается следующей главой. |

## Опциональные папки на книгу (не поставляются в этом шаблоне)

- `material/` — исходный/справочный материал при адаптации или исследовании (черновые
  заметки, собранные источники). В свободной форме; не читается ни одним workflow, пока вы
  на неё не укажете.
